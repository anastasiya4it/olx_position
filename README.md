# OLX Позиція — трекер позиції оголошення

Інструмент для перевірки на якому місці знаходиться ваше оголошення на OLX.ua у видачі за ключовими словами. ТОП/просувані оголошення не враховуються в рахунку.

## Як це працює

1. Ви вводите ключові слова, посилання на своє оголошення і місто
2. Бекенд запускає Chrome (Puppeteer), відкриває пошукову видачу OLX і гортає сторінки
3. Кожне ТОП-оголошення пропускається, звичайні рахуються по порядку
4. Коли ваше оголошення знайдено — повертається його абсолютна позиція

## Стек

- **Frontend** — Vue 3 + Vite + TypeScript
- **Backend** — Node.js + Express + Puppeteer Core (використовує системний Chrome)

## Встановлення

Потрібен [Node.js](https://nodejs.org) v18+ і встановлений Google Chrome.

```bash
# 1. Встановити залежності кореневого проекту
npm install

# 2. Встановити залежності фронтенду
npm install --prefix frontend

# 3. Встановити залежності бекенду
npm install --prefix backend
```

## Запуск

```bash
npm run dev
```

Відкриє одночасно:
- Фронтенд → http://localhost:5173
- Бекенд → http://localhost:3000

## Використання

1. **Ключові слова** — введіть запит, за яким хочете перевірити позицію (напр. `автошкола`, `iPhone 13 pro`)
2. **Посилання на оголошення** — вставте повний URL вашого оголошення з OLX, наприклад:
   ```
   https://www.olx.ua/d/uk/obyavlenie/avtoshkola-IDVYKtZ.html
   ```
3. **Місто** — виберіть зі списку (за замовчуванням Запоріжжя, є "Вся Україна")
4. Натисніть **Перевірити позицію**

Перший запит займає 5–10 секунд (запуск Chrome). Якщо оголошення не на першій сторінці — парсер гортає далі, до 20 сторінок (~800 оголошень).

## Де знайти Chrome якщо не запускається

За замовчуванням бекенд шукає Chrome тут:
```
C:\Program Files (x86)\Google\Chrome\Application\chrome.exe
```

Якщо Chrome встановлено в іншому місці — задайте змінну середовища:
```bash
set CHROME_PATH=C:\path\to\chrome.exe
npm run dev:backend
```
